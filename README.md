# CPU Monitor
## 简介
一款仿任务管理器的能够实时显示 CPU 利用率的 `electron` 桌面应用

## 运行方式

### windows 下
执行以下命令：
```
start electron.exe app
```
### macOS 下
下载 `electron-v4.0.3-mas-x64.zip`，解压后将 Electron.app 放入该项目的目录下，然后执行以下命令：
```
open Electron.app --args {app 文件夹的绝对路径}
```
